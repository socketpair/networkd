Identity-driven and Role-based Access Identify users before network access is allowed. Dynamically provision users based on roles.
Dynamic ACLs Dynamically respond to network events for faster containment of threats.
Gratuitous ARP Protection Protect network connections against man-in-middle attacks.
Source IP Lockdown Prevent network users from spoofing source IP addresses.
Rogue DHCP Server Detection Detect rogue DHCP servers installed by network users at the access edge.
MAC Address Limit/Lockdown Prevent unauthorized device connections (i.e. wireless access points, game consoles, etc...).
Protocol Anomaly Detection Detect malformed packets from malicious users.
